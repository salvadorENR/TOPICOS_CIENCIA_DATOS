=== REPRESENTACIONES ALGEBRAICAS ===

1. RED NEURONAL (5 neuronas) - Representación Conceptual:
   f(x) = σ(W2 * φ(W1 * x + b1) + b2)
   donde:
   - x = [failures, goout, schoolsup, famrel] (entradas normalizadas)
   - W1 = matriz de pesos capa oculta (4×5)
   - b1 = vector de sesgo capa oculta (5×1)
   - φ = función de activación (sigmoide/tanh)
   - W2 = matriz de pesos capa salida (5×1)
   - b2 = sesgo capa salida
   - σ = sigmoide para probabilidad final

2. SVM RADIAL - Representación:
   f(x) = sign( Σ[α_i * y_i * K(x_i, x)] + b )
   donde:
   - K(x_i, x) = exp(-γ * ||x_i - x||²) (Kernel RBF)
   - x_i = vectores de soporte
   - α_i = coeficientes del modelo
   - γ = parámetro del kernel
   - b = término de sesgo
